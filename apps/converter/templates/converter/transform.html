Clip
Cut input layer to a boundary polygon — e.g. Gujarat Roads → Surat Roads
Input layer is clipped to the exact extent of your boundary file. Features that cross the boundary are trimmed.
Click or drag your input layer here
Click or drag your boundary polygon here
Use Clip to get exact boundary geometry. Use Drop to keep only fully-contained features.
GeoPandas · gpd.clip()
Simplify
Reduce geometry vertices — e.g. 50,000 vertices → 500 vertices
Simplification removes unnecessary vertices while preserving overall shape. Higher tolerance = fewer vertices but less accuracy.
Click or drag your vector file here
In layer's CRS units. For EPSG:4326, ~0.0001° ≈ 11m.
Prevents self-intersections and collapsed geometries during simplification.
gdf.geometry.simplify(tolerance)
Buffer
Create distance zone around geometry — e.g. Road Centerline → 50m Buffer
For geographic CRS (EPSG:4326), buffer distance is in degrees. Reproject to a metric CRS first for accurate metre-based buffers.
Click or drag your vector file here
In CRS units (metres for projected, degrees for geographic).
Segments per quarter-circle. More = smoother but larger file.
gdf.geometry.buffer(distance)

Dissolve
Merge features sharing a common attribute — e.g. Surat + Navsari + Valsad → South Gujarat
Adjacent features with the same value in the dissolve field will be merged into a single geometry.
Click or drag your vector file here
Leave blank to dissolve all features into one.

Geodesic Calculations
Accurate Earth-surface area, length, and perimeter using WGS84 ellipsoid
geometry.area is NOT accurate on Earth! Use pyproj.Geod for geodesic area/perimeter — results are added as area_sqm and length_m fields.
Click or drag your vector file here
pyproj.Geod(ellps="WGS84").geometry_area_perimeter()
