ℹ️ About Raster Spike v1.0

Contingency feature for GIS file converter

📋 What is the Raster Spike?

A 2-week raster spike evaluates demand for raster ingestion before v1 ships. The spike covers GeoTIFF metadata extraction, reprojection, and basic conversion.

🎯 Testing Scope

  • GeoTIFF Metadata Extraction: Read resolution, CRS, bands, file size, projection information
  • Raster Reprojection: Convert between coordinate systems (e.g., EPSG:4326 → EPSG:3857)
  • Basic Raster Conversion: Convert GeoTIFF to Cloud Optimized GeoTIFF (COG)

📊 Expected Outcome

If the spike confirms real customer demand, basic raster onboarding ships in v1. If not, it moves to v1.5+.

💡 Supported Formats

  • GeoTIFF (.tif, .tiff)
  • Cloud Optimized GeoTIFF (.tif)
  • Other GDAL-supported raster formats
